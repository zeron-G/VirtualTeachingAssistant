# Reports

This directory is reserved for sanitized, reproducible engineering reports such
as static architecture inventories or public-safe evaluation summaries.

Do not commit generated runtime logs, local absolute paths, usernames, prompts,
student/course data, provider output, credentials, audit events, environment
snapshots, or incident evidence. Prefer regenerating reports in CI and publishing
only aggregate results with the producing command, commit, tool version, and
known limitations.
