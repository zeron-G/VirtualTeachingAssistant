# P1: Complete institutional identity and data-governance review

Category: Security / Data

Impact: Production use could expose student data or grant instructor actions to
an unverified identity if deployed before Carey/JHU controls are selected.

Evidence: The current repository has no institutional SSO, data retention,
records classification, or approved provider inventory.

Acceptance criteria: Approved SSO/RBAC, data-flow inventory, retention policy,
provider agreements, accessibility review, and documented production data
classes.

Verification: Institutional sign-off plus authorization and data-deletion tests.
